แนะนำการสร้างเว็บด้วย GitHub Pages

GitHub Pages คืออะไร?

GitHub Pages เป็นบริการฟรีที่ให้คุณสามารถโฮสต์เว็บไซต์ของคุณโดยใช้ GitHub Repository

วิธีใช้งาน GitHub Pages

  1. สร้าง GitHub Repository
  2. เพิ่มไฟล์ HTML, CSS และไฟล์ที่ต้องการ
  3. ไปที่ Settings ของ Repository และเปิด GitHub Pages
  4. เลือก Branch และกดบันทึก ระบบจะสร้าง URL สำหรับเว็บของคุณ

ตัวอย่างโค้ด HTML พื้นฐาน

<!DOCTYPE html>
<html>
<head>
    <title>My GitHub Page</title>
</head>
<body>
    <h1>สวัสดี GitHub Pages!</h1>
</body>
</html>
